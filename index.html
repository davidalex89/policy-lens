policy-lens

Evaluate security policies against compliance frameworks — or compare two policies side-by-side — using layered LLM prompts via Ollama.

ⓘ Getting started with Ollama click to expand

policy-lens runs entirely on your machine using Ollama for local LLM inference — no API keys, no data sent anywhere.

1
Install Ollama
Download from ollama.com/download and follow the installer for your OS.
2
Pull a model
Open a terminal and run one of: 
ollama pull gemma3        # recommended — fast, accurate
ollama pull llama3.2
ollama pull mistral
3
Start the Ollama server
ollama serve
Leave this terminal running. The status indicator next to the URL field below will turn ● green when connected.
4
Serve this page over HTTP
Browsers block requests from file:// pages. Serve index.html locally instead: 
python3 -m http.server 8080
Then open http://localhost:8080 in your browser. (If you are already seeing this page at a localhost URL, you can ignore this step.)
